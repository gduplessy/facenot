# Hot or Not?  

Simple Rails app using [facebook](http://www.facebook.com) profile pictures for a hot or not like fight.   
Goals? To learn:   
* FaceBook's API   
* More Rails   
* Algorithms   
   
by [Georges Duplessy](http://gduplessy.com)
